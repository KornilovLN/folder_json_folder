### Statistics of algorithms used in mnemo

source: /var/ekatra/tmp/ISO/IO-7.3-76.tar.gz

```
cd /var/lib/docker/volumes/sunpp-var-data/_data/tmp/u3/results/mnemos
grep -f *.mnemo  | grep "algor" | \
   awk -e 'match($0, /algor=['\'\"'][^'\'\"']*['\'\"']/) {print substr($0, RSTART+0, RLENGTH-0)}' | \
   sed "s/\"/'/g"| \
   sort | \
   uniq -c | \
   sort -bnr

```

```
    Переходит в указанный каталог.
    Ищет строки, содержащие слово "algor" в файлах с расширением .mnemo.
    Извлекает части строк, содержащие шаблон algor="..." или algor='...'.
    Заменяет двойные кавычки на одинарные.
    Сортирует строки.
    Удаляет дубликаты и подсчитывает количество каждого уникального вхождения.
    Сортирует результат по количеству вхождений в порядке убывания.

    В итоге, список всех уникальных строк, содержащих algor='...',
    отсортированный по убыванию количества вхождений.
```
